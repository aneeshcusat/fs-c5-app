Understanding Resource Planner

The Resource Planner provides a comprehensive view of resource allocation, helping you plan and balance workloads across projects and team members.

Accessing Resource Planner

Navigate to Project ‚Üí Resource Planner
Select date range and filters
Resource allocation view is displayed

Resource Planner Views

Timeline View

Shows resources on a timeline:

  • X-axis: Dates
  • Y-axis: Employees or Projects
  • Bars: Allocated hours
  • Color coding: Project or status

Resource View

Grouped by employee:

  • See each employee's workload
  • Identify over-allocation
  • Find available capacity

Project View

Grouped by project:

  • See resource needs per project
  • Identify resource gaps
  • Plan resource assignments

Using Resource Planner

Viewing Allocations

Select date range
Filter by teams or projects
View allocations on timeline

Identifying Over-Allocation

Look for:

  • Red indicators showing over-allocation
  • Hours exceeding capacity
  • Conflicting assignments

Finding Available Capacity

Identify:

  • Employees with available hours
  • Time periods with capacity
  • Under-utilized resources

Planning New Assignments

Find available resource slot
Drag and drop or create new assignment
Adjust hours and dates
Save assignment

Resource Capacity Settings

Configure employee capacity:

Open employee details
Set weekly capacity (hours per week)
Set availability dates
Save settings

Understanding Feedback

The Feedback system allows you to collect structured feedback on projects, helping improve processes and track satisfaction.

Accessing Feedback

Navigate to Project ‚Üí Feedback
Or open Project Details ‚Üí Feedback tab

Requesting Feedback

Open project details
Go to Feedback tab
Click "Request Feedback"
Select feedback template
Select recipients (team members, clients)
Set due date
Send feedback request

Submitting Feedback

Receive feedback request (email or notification)
Click link to open feedback form
Answer questions/rate items
Add comments
Submit feedback

Viewing Feedback

View collected feedback:

  • Summary statistics
  • Individual responses
  • Comments and suggestions
  • Trends over time

Feedback Templates

Organizations may have predefined feedback templates:

  • Project completion feedback
  • Team performance feedback
  • Client satisfaction surveys
  • Process improvement feedback

üè¢ Accounts Management

Understanding Accounts

Accounts represent clients or business entities. Projects are associated with accounts for organization and reporting.

Accessing Accounts

Navigate to Project ‚Üí Accounts
List of accounts is displayed

Viewing Accounts

Account list shows:

  • Account name
  • Account code
  • Contact information
  • Number of projects
  • Total value
  • Status

Creating Accounts

Click "CREATE ACCOUNT"
Fill in account information:
  • Account name (required)
  • Account code
  • Contact person
  • Email
  • Phone
  • Address
  • Billing information
Click "Create"

Editing Accounts

Find account in list
Click "Edit"
Update information
Save changes

Viewing Account Details

Click on account name to see:

  • Full account information
  • Associated projects
  • Contact history
  • Billing information
  • Account notes

Account Projects

View all projects for an account:

Open account details
Go to Projects tab
See all projects associated with this account

üîç Advanced Search

Global Search

The search bar in the header allows you to search across:

  • Projects
  • Tasks
  • Employees
  • Accounts
  • Files

Using Global Search

Click in the search box (top header)
Type your search term
Results appear as you type
Click on a result to open it

Dedicated Search Page

Navigate to Project ‚Üí Search
Use advanced search filters:
  • Search by type (Project, Task, etc.)
  • Date range
  • Status
  • Team
  • Custom fields
Enter search terms
Click "Search"

Search Operators

Some search fields support operators:

  • Quotes: Exact phrase ("project name")
  • AND: All terms must match
  • OR: Any term can match
  • NOT: Exclude terms

Saved Searches

Save frequently used searches:

Create search with desired filters
Click "Save Search"
Name the search
Access saved searches from dropdown

üìÖ Calendar View

Understanding Calendar

The Calendar view provides a visual timeline of projects, tasks, and activities.

Accessing Calendar

Navigate to Project ‚Üí Calendar
Calendar view is displayed

Calendar Views

Month View

Shows monthly calendar grid:

  • Projects and tasks on their dates
  • Color coding by type or status
  • Click date to see details

Week View

Shows weekly timeline:

  • More detailed view
  • See task durations
  • Better for planning

Day View

Shows single day in detail:

  • All activities for the day
  • Time breakdown
  • Detailed schedule

Filtering Calendar

Filter calendar by:

  • Projects
  • Teams
  • Task status
  • Employee

Creating Items from Calendar

Click on a date
Select "Create Task" or "Create Project"
Date is pre-filled in form
Complete creation
