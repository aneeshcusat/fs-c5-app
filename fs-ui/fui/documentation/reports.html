Employee Utilization Report

Shows how employees are utilizing their time across projects.

Generating Employee Utilization Report

Navigate to Project ‚Üí Reports
Select "Employee Utilization" from report dropdown
Set date range
Select teams/employees to include
Set utilization limits (lower and upper) if needed
Click "Generate Report" or "View"

Report Contents

The report shows:

  • Employee name
  • Total hours worked
  • Billable vs non-billable hours
  • Utilization percentage
  • Breakdown by project
  • Comparison to targets

My Utilization Report

Personal utilization report showing your own time utilization.

Select "My Utilization" report
Set date range
Generate report

Weekwise Employee Utilization Monthly

Shows weekly breakdown of employee utilization for a month.

Select "Weekwise Employee Utilization Monthly"
Select month
Select teams/employees
Generate report

Utilization by Skills

Shows utilization broken down by employee skills.

Select "Utilization By Skills"
Set filters
Generate report

Notifying Defaulters

For Employee Utilization report, you can notify employees who fall below utilization thresholds:

Generate Employee Utilization report
Set lower limit threshold
Click "Notify Defaulters" button
System sends email notifications to employees below threshold

üì• Exporting Reports

Export Formats

Reports can typically be exported in:

  • Excel (.xlsx): For data analysis
  • PDF: For sharing and printing
  • CSV: For importing into other systems

How to Export

Generate the report (view it first)
Click "Export" button
Select export format (Excel, PDF, CSV)
File downloads automatically

Export Options

Some reports offer export options:

  • Include/exclude specific columns
  • Format options (colors, borders)
  • Page orientation (for PDF)
  • Date format

Emailing Reports

Some reports can be emailed directly:

Generate report
Click "Email Report" button
Enter recipient email addresses
Add subject and message
Select format (Excel, PDF)
Click "Send"

üìÖ Scheduled Reports

Creating Report Configurations

Save report configurations for easy reuse:

Generate a report with your desired filters
Click "Save Configuration" or "Create Report Config"
Enter configuration name
Save the configuration

Using Saved Configurations

Go to Reports page
Select report type
Select saved configuration from dropdown
Filters are automatically applied
Generate report

Scheduling Reports (If Available)

Some organizations may have scheduled report functionality:

Create report configuration
Click "Schedule" option
Set schedule frequency (Daily, Weekly, Monthly)
Set recipients
Save schedule
