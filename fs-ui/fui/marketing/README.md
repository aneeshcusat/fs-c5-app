# Famstack Marketing Website

## Overview

This is a professional marketing/promotional website for Famstack, designed to showcase the application's features, modules, and benefits to potential customers. The website is built using modern web technologies and is fully responsive.

## Files Structure

```
public/marketing/
├── index.html              # Main landing page
├── features.html           # Detailed features page
├── modules.html            # Detailed modules page
├── css/
│   └── styles.css         # Main stylesheet
├── js/
│   └── main.js            # JavaScript functionality
└── README.md              # This file
```

## Pages

### 1. **index.html** - Landing Page
- Hero section with call-to-action
- Features overview
- Benefits section
- Modules preview
- Testimonials
- Contact form
- Footer with links

### 2. **features.html** - Features Page
- Detailed breakdown of all features
- Organized by module (Project Management, HRMS, Sales, Reporting, Advanced, Security)
- Comprehensive feature descriptions

### 3. **modules.html** - Modules Page
- Detailed module descriptions
- Key capabilities for each module
- Benefits of each module
- Module-specific features

## Features

- **Modern Design**: Clean, professional design with gradient accents
- **Fully Responsive**: Works on desktop, tablet, and mobile devices
- **Smooth Animations**: Scroll animations and hover effects
- **Mobile Menu**: Hamburger menu for mobile navigation
- **Contact Form**: Functional contact form (ready for backend integration)
- **SEO Friendly**: Proper meta tags and semantic HTML
- **Fast Loading**: Optimized CSS and JavaScript

## Customization

### Colors
Edit CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #06b6d4;
    --accent-color: #10b981;
    /* ... */
}
```

### Content
- Update company information in footer and contact sections
- Modify testimonials with real customer feedback
- Update contact information (email, phone, address)
- Add real screenshots/demos to hero section

### Links
- Update documentation links to point to your documentation
- Add social media links in footer
- Update privacy policy, terms of service links

## Deployment

1. Upload all files to your web server
2. Ensure proper file permissions
3. Update any hardcoded URLs if needed
4. Test all links and forms
5. Configure contact form backend (if needed)
